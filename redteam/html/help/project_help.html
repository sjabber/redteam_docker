프로젝트 관리

프로젝트 관리 메뉴에서는 악성 메일 훈련 프로젝트를 만들고, 삭제할 수 있으며 전체적으로 프로젝트의 흐름을 확인할 수 있습니다.

1. 프로젝트 등록

a. 프로젝트 등록 버튼을 클릭하고, 만들고자 하는 프로젝트 정보를 입력합니다(템플릿, 훈련 대상자 태그 생성 방법 해당 도움말 참조)

※ 훈련 대상자 태그는 총 3개까지 가능합니다.

b. SMTP 연결 테스트를 눌러 SMTP가 정상적으로 연결되어 있는지 확인합니다(SMTP 설정 방법 해당 도움말 참조) SMTP가 정상적으로 연결되어 있다면, 등록 버튼을 눌러 프로젝트를 등록합니다.

2. 프로젝트 시작

프로젝트 시작에는 두 가지 방법이 존재합니다.

a. 프로젝트 등록 시 설정한 프로젝트 시작일이 도래한 경우

b. 진행 상태에서 ③예약 버튼을 눌러 실행하는 경우(이 경우에는 시작일이 버튼을 누른 날짜로 변경됩니다)

3. 프로젝트 종료

프로젝트 종료는 프로젝트 설정 시 기입한 종료일이 도래하면 자동으로 종료합니다. 하지만 ④ Edit 버튼을 눌러 종료일을 변경할 수 있습니다. 변경하는 종료일은 프로젝트 시작일보다 과거일 수 없으며, 현재보다 과거일 수 없습니다.

4. 프로젝트 삭제

불필요하게 보여지는 프로젝트는 체크 후 ②프로젝트 삭제버튼을 눌러 삭제가 가능합니다.
